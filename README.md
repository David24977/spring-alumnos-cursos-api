# API REST - Gestión de Alumnos y Cursos

Proyecto de práctica con Spring Boot en el que se implementan las entidades Alumno y Curso, con relación ManyToOne.

## Funcionalidades

- Crear curso
- Crear alumno
- Listar cursos
- Listar alumnos
- Buscar alumnos por nombre
- Listar alumnos de un curso
- Cambiar un alumno de curso (PATCH)
- Actualizar y eliminar alumnos y cursos

## Tecnologías

- Java 17
- Spring Boot 3
- MySQL
- JPA/Hibernate
- DTOs (Request/Response)
- Swagger / OpenAPI
- Maven
